Programa de recompra no WhatsApp para delivery

Cliente que já pediu uma vez é o mais barato de fazer pedir de novo. É só falar com ele na hora certa.

Resposta curta: um programa de recompra no WhatsApp usa a base de clientes que você já tem (nome, o que pediram, quando) para trazer o cliente de volta com a mensagem certa na hora certa: um lembrete de recorrência, uma oferta no dia mais fraco, um "sentimos sua falta" para quem sumiu. É pedido novo sem pagar de novo pela aquisição, porque o cliente já é seu.

Por que recompra vale mais que só buscar cliente novo

Trazer um cliente novo custa anúncio toda vez. Fazer um cliente que já gostou pedir de novo custa uma mensagem. As duas coisas importam, mas a maioria dos deliveries só investe na primeira e esquece a segunda, que é a mais barata e a mais lucrativa. Quem pede uma vez e é bem atendido tem tudo para virar recorrente.

O que você precisa para fazer recompra

Uma coisa só: a base de clientes. E aqui mora o detalhe que muito dono de delivery não percebe. No iFood, o cliente é do app: você não tem o contato para chamar de volta. No seu canal próprio, com o pedido caindo no WhatsApp, você fica com o nome e o histórico e pode falar com ele quando quiser. É por isso que construir o canal próprio é o que destrava a recompra.

As 3 campanhas de recompra que funcionam

  1. Lembrete de recorrência. Quem pede toda sexta recebe um "bora repetir?" na sexta de manhã. Simples e certeiro.
  2. Oferta no dia parado. Naquele dia da semana mais fraco, uma condição especial para a base enche a cozinha.
  3. Resgate de inativo. Quem não pede há 30 ou 60 dias recebe um "sentimos sua falta" com um empurrãozinho para voltar.

Cadência: falar na hora certa, sem encher o saco

Recompra não é disparar promoção todo dia para todo mundo, isso queima a base e vira bloqueio. O jogo é mandar a mensagem certa para a pessoa certa: quem é recorrente, quem sumiu, quem pediu ontem. Segmentar é o que separa recompra de spam.

Como a IA cuida da recompra sozinha

Fazer isso na mão dá trabalho: lembrar quem sumiu, quem pede toda semana, quem nunca voltou. Uma IA de atendimento registra cada cliente e a última compra e dispara a mensagem certa no momento certo, sozinha. A mesma Secretária IA que atende e recupera o carrinho também organiza a recompra, para o cliente pedir de novo sem você precisar lembrar.

Perguntas frequentes

O que é recompra no delivery?

É fazer o cliente que já pediu voltar a pedir. Como ele já conhece e gostou do seu delivery, o custo de trazer esse pedido de volta é muito menor do que conquistar um cliente novo do zero. A recompra é feita falando com a sua base de clientes na hora certa, em geral pelo WhatsApp.

Como fazer o cliente pedir de novo?

Falando com ele no momento certo: um lembrete para quem pede toda semana, uma oferta no dia mais parado e um resgate para quem sumiu. O segredo é ter a base de clientes (nome e histórico) e mandar a mensagem certa para cada perfil, sem disparar promoção para todo mundo.

Com que frequência mandar mensagem de recompra?

O suficiente para lembrar, sem cansar. Mensagem demais queima a base e vira bloqueio. O ideal é segmentar: quem é recorrente recebe no ritmo dele, quem sumiu recebe um resgate pontual. Falar na hora certa vale mais do que falar toda hora.

Dá para automatizar a recompra pelo WhatsApp?

Dá. Uma IA de atendimento registra cada cliente e a última compra e dispara sozinha a mensagem certa: lembrete, oferta em dia parado ou resgate de inativo. Assim a recompra acontece sem depender de alguém lembrar de olhar a base todo dia.
